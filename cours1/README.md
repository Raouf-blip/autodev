## Lancer les projets

### API PHP
```bash
cd immo-api-php
docker compose up -d --build
```
http://localhost:8080

### Client Vue
```bash
cd immo-client-vue
docker compose up -d
```
http://localhost:3000

### Admin React
```bash
cd immo-admin-react
docker compose up -d
```
http://localhost:3001
