# Immo Admin React

Interface d'administration pour la plateforme immobilière (React + Vite + Tailwind).

## Prérequis

- Docker et Docker Compose

## Installation

```bash
docker compose up -d
```

L'app est accessible sur http://localhost:3001

## Commandes utiles

```bash
docker compose up -d      # Démarrer
docker compose down       # Arrêter
docker compose logs -f    # Logs
```

## Sans Docker

```bash
npm install
npm run dev
```
