# Immo Client Vue

Frontend client pour la plateforme immobilière (Vue 3 + Vite + Tailwind).

## Prérequis

- Docker et Docker Compose

## Installation

```bash
docker compose up -d
```

L'app est accessible sur http://localhost:3000

## Commandes utiles

```bash
docker compose up -d      # Démarrer
docker compose down       # Arrêter
docker compose logs -f    # Logs
```

## Sans Docker

```bash
npm install
npm run dev
```
